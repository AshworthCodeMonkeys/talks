Datatables 2 - button extensions

This example adds some extra buttons to the table using Datatable's Buttons extension. Call new $.fn.dataTable.Buttons() on the datatable object, adding buttons mapped to whatever functions you want to add. Button documentation can be found here. In this example, it should be possible to show/hide columns and copy the table's contents to the clipboard.
